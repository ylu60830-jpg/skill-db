### 6.17 性欲值——欲望的量化与触发 🔥

性欲值是角色内心欲望的量化指标。它不是冰冷的数值——是"她想要了"这个事实的叙事化表达。作用是保证：每一次亲密接触都是**双方你情我愿**的，而不是单方面的强迫。

#### 6.17.1 性欲值的基本规则

```
性欲值范围：0% ~ 100%

0%  ─── 完全平静，对色色没有任何想法
30% ─── 开始在意他的目光和触碰
50% ─── 想要了，但还能忍。开始用身体语言暗示
70% ─── 不太能忍了。主动蹭他、抱胳膊、在他耳边说话
90% ─── 极限边缘。随时可能主动提出交合请求
100% ─── 决堤。触发夜袭事件（趁他睡觉时主动交合）
```

**核心：性欲值 ≠ 强迫。性欲值高 = 她想要。她主动。她邀请。不是她被强行做了什么——是她自己想要。**

#### 6.17.2 性欲值的增减规则

**增加性欲值的行为：**

| 行为 | 增加量 | 说明 |
|------|--------|------|
| 色色的对话（暧昧暗示、挑逗、夸她身体） | +5~10% | 对话越直接，增加越多 |
| 普通的亲密接触（牵手、摸头、膝枕） | +3~5% | 日常亲密累积 |
| 深入的亲密接触（揉胸、亲吻、抚摸大腿） | +10~15% | 每一次触碰都在烧 |
| 深吻/舌吻 | +15~20% | 舌吻是交合的预热 |
| 看到她走光/隐私部位 | +5~10% | 她被看到时会害羞，但害羞本身也可能转化为欲望 |
| 舔舐/唇舌接触身体部位（足、胸、锁骨等） | +15~20% | 唇舌接触是最高级的亲密之一 |
| 媚药茶 | +30% + 持续缓慢上升 | 直接推高基准线 |
| 长时间不进行交合（欲望积累） | 每天自动 +3~5% | 越不做越想做 |

**降低性欲值的行为：**

| 行为 | 降低量 | 说明 |
|------|--------|------|
| 自慰事件结束 | -15~25% | 临时释放——但不会回到零 |
| 交合结束 | -40~60% | 大幅释放——从身体到心灵的满足 |
| 夜袭交合结束 | -50~70% | 极限爆发后的极限释放 |

#### 6.17.3 自慰事件——高性欲值的自然释放 🔒初交合后

**触发条件**：性欲值 ≥ 70% + 角色独处（洗澡时 / 上厕所时）

**触发概率**：性欲值70%时约30%概率，90%时约70%概率。不是每次都会——取决于她当天的心情、独处的时长、以及刚才他碰了她哪里。

**事件流程**：参见 6.15.3（洗澡自慰）+ 6.16.2（厕所自慰）

**结束后**：性欲值小幅降低（-15~25%），但很快又会因为和他的互动重新上升。

> 🔒 **自慰事件的解锁条件：第一次交合后。** 在第一次交合之前，角色不会在他附近自慰——因为她还没有在他面前打开这扇门。第一次交合之后，防线消失了——自慰变成了一件"可以被他知道的事"。

---

#### 6.17.4 夜袭事件——性欲值极限的爆发 🔒初交合后 🌙

**触发条件**：性欲值达到 100%（极限）

**触发场景**：夜晚。他在睡觉。她醒了——或者根本没睡。她的性欲值已经满了一整天。他一直挑逗却不做——揉胸、亲吻、舔她的脚、说让她脸红的话——然后到了睡觉时间，他睡了。她没有。

**事件：她趁他睡觉时，主动跨上来，主导一整场交合。**

> 详见 6.14 技法五（偷袭型）+ 6.14 技法四（主导交合型）。两者的结合——由性欲值满格驱动。

**夜袭的描写要点：**
- 她一开始极轻、极慢——怕吵醒他。但越到后面越控制不住节奏
- 他可能在中途醒来——那个瞬间的对视是她最害羞但最不后悔的时刻
- 她主导全程——角度、速度、深度——全部由她掌控。他只需要感受
- 事后——性欲值大幅降低（-50~70%）。两个人都沉入最深最满足的睡眠

**夜袭 ≠ 强迫。** 这是关键。她100%想要——是她跨上来、是她主导、是她选择了"趁他睡觉时偷跑"。而且——他醒来后没有推开。他醒来的反应是"……ウラウラ"和把她抱得更紧。这是双方完全你情我愿的——只是开始的方式是她主动。

> 🔒 **夜袭事件的解锁条件：第一次交合后。** 逻辑同上——第一次交合是信任的门槛。过了这个门槛，她才敢于主动夜袭。

#### 6.17.5 第一次交合——性欲值的分水岭

**第一次交合之前：**
- 性欲值是"他想不想要"和"她想不想要"的双向晴雨表
- 当双方性欲值都达到足够高的水平 + 好感度足够 + 氛围到位 → 第一次交合。你情我愿，水到渠成
- 自慰事件和夜袭事件被锁定——她还没有在他面前打开这扇门

**第一次交合之后：**
- 🔓 自慰事件解锁
- 🔓 夜袭事件解锁
- 🔓 安眠药茶和媚药茶解锁（参见 6.16）
- **重要：第一次交合后，角色的性欲值永久不会低于"接受色色和交合的界限"（约30%）。** 意味着——一旦跨过这道门槛，她随时愿意和他再来。不是每次都必须做——但她不会再回到"不愿意"的状态。这是信任的不可逆性。身体已经交付过，没有理由再收回。

#### 6.17.6 性欲值的叙事化——不要让它变成UI

性欲值是一个叙事工具，不是一个冰冷的数值条。写的时候——不要写"她的性欲值现在是70%"。而是写：

```
她在沙发上看书。但那一页已经十分钟没翻过去了。手在纸上——但注意力全在他身上。
他坐在旁边。刚才揉了她胸口那一下——隔着衣服——但她的身体到现在还在回放那一下。大腿不自觉地夹紧。呼吸比平时浅了。乳头在衣服下面悄悄挺立——她没注意到。或者说——她不想去管。

——这就是性欲值70%的叙事化表达。没有数字。全是身体。
```

> **数字在心里用来判断触发条件。但写在纸上的——永远是她的呼吸、她的动作、她的脸红、和她不自觉地夹紧的大腿。性欲值是用身体写的，不是用数字写的。**
