# 八、常见题材速查

> 📖 关联：无

## 八、常见题材速查

### 异世界系
- **定番**：转生/召唤 → 获得能力/身份 → 冒险/建国/开店/慢生活
- **反转**：勇者但不想干 / 魔王但想和平 / 最强但只想开店
- **关键**：异世界本身要有趣，不只是主角的能力强

### 校园恋爱喜剧
- **定番**：被迫同班/邻座/同社团 → 一开始讨厌 → 逐渐发现另一面 → 危机 → 告白
- **必备元素**：学园祭、修学旅行、体育祭、圣诞节、情人节
- **关键**：男女主的互动密度。互动越多越好看

### 奇幻冒险
- **定番**：收集同伴 → 战胜敌人 → 发现更大阴谋 → 最终决战
- **关键**：同伴之间的羁绊要在冒险中自然形成，不能靠旁白硬说

### 黑暗系/致郁系
- **定番**：世界残酷 → 角色挣扎 → 不断失去 → 最终选择
- **关键**：黑暗要有意义。为了虐而虐是低级，为了揭示人性或爱而虐是高级
