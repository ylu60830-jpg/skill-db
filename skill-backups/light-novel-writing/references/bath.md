# 6.15 浴室场景

> 📂 触发条件：写浴室/共浴/搓澡/泡澡/偷窥洗澡
> 📖 关联：intercourse.md, exposure.md

### 6.15 浴室场景——水汽中的亲密 🛁

日漫定番中的定番。第一次交合之后，一起洗澡是天经地义的事——身体已经交付过了，防线已经不存在了。浴室只是把这份亲密放在水汽里再浸泡一遍。

浴室场景分三大板块：**淋浴搓澡** → **泡澡** → **偷窥/偷内裤**。

---

#### 6.15.1 淋浴篇——搓澡与擦枪走火 🚿

**前提**：两人已经在浴室里。可以是一起进来的（交合后），也可以是一个先洗另一个后进来。淋浴的核心道具：花洒的热水、泡沫、浴花/毛巾、搓背。

---

**A. 她给他搓澡——从后背到前胸，从上面到下面**

他背对着她坐着。她用浴花打泡沫，从肩膀开始搓。后背、肩胛骨、脊椎沟——每一寸。然后让他转过来。搓前胸。

搓着搓着——某一刻。他硬了。那东西在水流中挺起来——和她搓澡的动作发生在同一秒。

**不同性格的反应：**

**害羞系——手停住了，但没有移开。**
```
她的手指停在我胸口。浴花从手里滑掉了。

她当然看到了。花洒的水还开着一一温水从我的胸口往下淌，淌过腹部，淌过——那里。硬挺的、在水流中微微跳动的、完全暴露在她视线里。
「……ごめん。」
「……謝らないで。」
她的声音很小。但她的手从我的胸口往下滑——不是搓澡的动作。是抚摸。
「……手伝ってほしい？」
不是"要不要做"。是"要不要我帮你"。主语是我，动词是帮。她把主动权留给我。
「……ほしい。」
她的手指穿过水流——轻轻圈住了那里。指腹搓澡时还是粗糙的，碰到这里却像换了个人——极轻、极柔。
「……お返し、してもらうからね。」
——这家伙，害羞到耳根红透，却已经开始计划下一轮了。
```

**小恶魔——发现了之后笑一下，搓澡升级。**
```
「……あら。」
她的声音从背后传来。浴花还在我背上，但她的另一只手——已经绕到了前面。
「師匠のここ——さっきまであんなに頑張ったのに、まだ元気なんスね。」
手指没碰到——但在水流下，那个指腹的影子和我的那里只隔着一层水膜。
「……いいよ。手、動かして。」
她在我耳边说完这句话，在我脖子上轻轻咬了一口。
——搓澡结束。第二回合开始。原地。
```

**傲娇——"笨蛋！变态！……但我帮你。"**
```
「——なっ！」
她手里的浴花掉在地上。从我背后能看到她耳根变成了粉色。
「な、なんでこんな時に——バカ！変態！」
「生理現象だ——」
「分かってるけど！でも——」
她沉默了一下。然后手指从背后伸过来——没有圈住，只是用指背轻轻碰了一下。像在确认温度。
「……自分でやるより、私がやったほうが——気持ちいいか。」
不是疑问句。说完她就把手绕到了正面。脸还贴在我后背上——不敢看。
「褒めて。私の手。ちゃんと気持ちよくさせるから。」
```

**B. 他给她搓澡——他掌握主导权**

他给她搓背。从肩膀到脊椎到腰——水流冲走泡沫，露出被搓红的皮肤。他搓得越久，她的呼吸越不稳。然后他的手滑到了前面——从背后环住她。

搓她的胸口时——不是为了搓澡。是为了感受那片柔软在泡沫中更加光滑的触感。隔着滑溜溜的沐浴露，手指陷进去的深度——比平时多了三成。

```
她的背靠在我胸口。花洒的水从我们头上浇下来。

我把浴花换成手掌。掌心贴着她的锁骨下方——指尖抄起一堆泡沫。然后往下滑。泡沫像润滑剂。掌心下的皮肤更滑了。更热了。
滑到那片柔软。手指陷进去——不是揉。是搓。搓澡的手法。但隔着这层泡沫，每一次"搓"都变成了"揉"。每一次"揉"都让她往我怀里更靠一寸。

「……せっけん、まだ落ちてないっスよ。」
「分かってる。」
「じゃあ——ちゃんと洗ってほしいっス。」
但她自己的手覆在我的手背上——把我的手指往更深处按了按。
——嘴上说"洗って"，身体在说"触って"。
```

#### 6.15.2 泡澡篇——同一缸热水里的狭小空间 🛀

两人同时坐进浴缸。热水漫到胸口。腿贴着腿。屁股贴着浴缸底——空间太小了。不需要刻意靠近，已经是最近的距离。

**泡澡的色气要素：**
- 热水让皮肤变粉——她的肩膀、锁骨、脸颊，全泛着淡粉色
- 水面下看不到的身体——腿怎么放的、贴在哪里了——只能靠触觉感知
- 蒸汽模糊了视觉→其他感官放大：水温、她皮肤的滑腻、洗发水的味道
- 水面晃动——任何一个小动作都会让水面荡开波纹，暴露水下的动作

```
浴缸太小了。两个人挤进来，腿只能交叠着放。她的大腿压在我的腿上——水下。热水的浮力让她的身体微微往上飘。每一次呼吸，她的膝盖就在我大腿内侧轻轻蹭过。

水面在蒸汽里微微晃动。看不清水下——但感觉得到。她的脚——不知什么时候贴在我的小腿上。足底。柔软的、被热水泡热的足底。沿着我的小腿缓缓往上滑。

「……浴槽、狭いっスね。」
她说着，眼睛没有看我。但她的脚已经滑到了我的大腿——在往更深处探。
水面晃了一下。不是因为浴缸太小。
```

**泡澡→擦枪走火：**

他在水下硬了。浴缸太小藏不住。她感觉到了——顶在她大腿上的硬度。

```
「……あ。」
她的膝盖轻轻碰了一下那里。不是故意——是太挤了。但碰到之后她没有移开。水面晃了一下。
「……分かるっス。だって——アタシも。」
她从水下抓住我的手——拉到她的大腿内侧。皮肤在热水里更滑了。不是大腿。是更深处——她的手指引着我的手指，碰到了那片柔软的核心。
水面下——水是热的，她是更热的。
「……お湯の中ですると——気持ちいいらしいっスよ。」
```

#### 6.15.3 偷窥篇——她在浴室里，他在门外 👁️

她去洗澡。单独去的。他听到水声——然后脑子里开始想。门可能没锁。或者有一道缝。或者门上有磨砂玻璃——看不清，但能看到模糊的轮廓。

**撞见自慰：**

他从门缝往里看——她不是在正常洗澡。一只手撑在墙上，另一只手——在下面。花洒的水从她背上流下来，盖不住从她喉咙里漏出来的闷哼。

**选择一：闯进去。**
```
我推开门。她猛地转头——脸上全是水珠和红晕。手还没来得及从下面抽出来。眼睛瞪大看着我——不是恐惧。是被发现了的震惊。
「……いつから。」
「さっきから。」
沉默。花洒还在流水。她的手指还卡在那里——没有抽出来。不是忘了。是没有必要了。她看着我。我看着她的手。花洒的水从两人之间流过。
「……見てるだけ？それとも——」
她的手指——慢慢地——在自己那里动了一下。当着我的面。
「……手伝ってくれる？」
```

**选择二：继续偷窥——他自己也加入。**
```
我从门缝看到的一切——她的手在下面前后摩擦、腰微微弓起、膝盖在瓷砖上轻轻打滑。听着她压得极低极低的"ん……ん……"。

我的手也放到了自己下面。从外面——透过那道门缝。她的动作越来越快，我的也是。她在里面达到顶点的那一声漏出来的呜咽——和我在外面同时闷住的喘息。隔着一道门。完全同步。

然后我靠在墙上喘气。她关掉花洒。水声戛然而止。

「……そっちも、終わった？」
——她知道。从头到尾。她都知道我在门外。
```

#### 6.15.4 偷内裤篇——她刚脱下来的体温 🩲

她去洗澡。换下的内裤搭在更衣篮里。还温热。不是被烘干机烘出来的热——是被她体温焐了一整天的热。

**三种用途：嗅/舔、自慰、被发现的三种结局**

| 用法 | 描写要点 | 色气来源 |
|------|---------|---------|
| **嗅** | 凑近鼻尖——洗衣液的清香+她体香+残余的体温 | 这是"她的一天"被锁在布料里的味道 |
| **舔** | 舌尖碰到裆部的布料——微咸、微涩、棉布在口中从干变湿 | 不是舔布——是舔她留在布上的痕迹 |
| **自慰** | 把内裤裹在硬挺的那里——布料还温热——用手握着上下 | 她的内裤贴着她最私密的地方，现在贴着他最私密的地方 |

**被发现的三种结局：**

**结局一：骂完就没事型（低好感度/傲娇）**
```
「——何してんのよ、この変態！！ド変態！！ロリコン！！雑魚！！」
她把内裤从我手里一把夺走。整张脸红透了——分不清是怒还是羞。抱在怀里。内裤抱在胸口。然后——沉默了。
「……嗅いだ。」
「……嗅いだ。」
「……どんな匴いだった。」
「……お前の匂い。」
她愣了一秒。然后内裤砸在我脸上。「勝手に人の下着で興奮してんじゃねー！！」转身冲进浴室。但没锁门。
```

**结局二：害羞遮体型（中好感度）**
```
推开更衣室门——她站在门口。刚从淋浴间出来，浴巾裹着身体。然后看到了我手里的——她的内裤。还叠着。还没被亵渎——只是被拿着。
「……あ、あの——」
她把浴巾往上拉了拉。耳根烧起来了。但没有抢回去。只是看着那条内裤在我手里。
「……返して、ほしい。けど——」手指揪紧浴巾，「——そんなに気になるなら、洗ったの——あげる。今度。洗ってから。」
——不是拒绝。是延迟交付。她把"给你我的内裤"当成了一个约定。
```

**结局三：邀请升级型（高好感度/小恶魔/温柔系）**
```
她站在浴室门口。浴巾松松地裹着。头发还在滴水。然后她看到了——我贴在自己脸上的、她刚脱下的内裤。

沉默了好几秒。我僵住了。要挨骂了。但她没有骂。她走过来——在我面前蹲下。把我的手指从内裤上拿开。然后她用自己的手指——勾住那条内裤——把它从我脸上揭下来。

「……直接嗅いたかったの。」
她把内裤放在一边。然后拉住我的手——拉向她自己。不是浴巾。是浴巾下面。大腿内侧。刚洗完澡的皮肤又热又滑。
「こっちのほうが、温かいでしょ。」
——不是骂我变态。是给了我一整个她。
```

#### 6.15.5 偷内裤进阶——她用内裤帮他弄 👋

这是结局三的升级版。她发现他拿了自己的内裤自慰——没有生气，反而主动用那条内裤套住他那里，隔着内裤的布料用手帮他。

```
她看到我手里那条——她刚脱下的白色棉质内裤。叠得歪歪扭扭。裆部中央有一小片被唾液染湿的深色。

没有生气。没有骂人。只是轻轻"あーあ"地叹了口气。
然后她从我手里拿走那条内裤。把它翻到正面。把裆部对准我的那里——那层薄棉布——盖了上去。包住。

「噛んだとこ——ここ。」
她的手指隔着内裤按在顶上。那层布料——她穿过一整天的、裆部位置——现在贴在我最敏感的地方。
然后她的手指开始动。不是直接触碰。是隔着那层棉布——她自己的内裤——上下摩擦。棉布在龟头和手指之间发出细微的沙沙摩擦声。粗糙的纤维×她指腹的柔软×她裆部残留的体温。
「……直接じゃなくてごめん。でも——」
拇指在布料的顶端轻轻抹了一圈。棉布被先走汁濡湿了一小片——和她裆部原有的淡淡痕迹混在一起。
「——これで我慢して。」
```
