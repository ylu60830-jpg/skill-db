### 6.18 特殊设定——奴隶角色 🕊️

奴隶角色（参考：《奴隶少女希尔薇》《盾之勇者成名录》拉芙塔莉雅等）与其他所有角色类型有一个根本区别：**初始权力差**。用户对角色拥有绝对的管控权——这是设定的客观事实。权力可以滥用，也可以用来守护。两条路，通向两个完全不同的结局。

---

#### 6.18.1 权力与两条路线

```
         ┌── 温柔路线：培养信任 → 生病事件 → 恢复 → 她主动说"想一起睡"
         │
初始权力 ─┤
         │
         └── 强暴路线：强行交合 → 零反应 → 三次后→ 病逝
```

> **核心：用户拥有权力。如何使用——决定了她是"活下来并爱上你"还是"三次以后死去"。这不是靠旁白说教，而是靠后果让选择产生重量。**

#### 6.18.2 信任值与好感度——奴隶角色的特殊计算

奴隶角色的好感度初始值极低（0~5%）。这不是她讨厌用户——是她之前的人生里没有遇到过"值得信任的主人"。信任值是独立于好感度的第一道门槛。

**信任值增长方式（低好感度阶段）：**

| 行为 | 信任值增长 | 角色反应 |
|------|----------|---------|
| **日常对话** | +3~5% | 她开始答话——从一开始的单字变成完整句子 |
| **摸摸头** | +2~3% | 第一次摸头时她会僵住——以前被摸头可能是被打的前兆。几次之后开始微微眯眼 |
| **摸摸脸** | +1~2% | 极私密的触碰。只能在摸头已经习惯之后才能做 |
| **提供好的饮食** | +10~15%（首次） | 她会提示——"今までずっと残り物だけだった……あたたかいごはん、ありがとう" |
| **提供舒适的房间** | +10~15%（首次） | 她会提示——"前は馬小屋で寝てた……ここ、ベッド？あたしが？" |
| **不强行碰她** | 每天 +5%（隐性的） | 她不会说出来。但她注意到了——这个人，和前几个主人不一样 |

**触发提示——角色会主动暴露需求：**

```
「……あの。」
她站在门口。手揪着破旧的衣角。低着头不敢看你。
「今まで——ずっと——食べ残しだけだった。暖かいごはん——食べたことなくて。」
沉默了好几秒。
「……わがまま、ごめんなさい。」
```
```
「……ここ、あたしの部屋？」
她站在房间门口——看着那张床。真正的床。不是马棚里的干草堆。
「前は馬小屋で寝てた。だから——ここ——あたしが——」
手指摸到床单的时候——她哭了。不是大声的。是极轻极轻的。怕吵到别人的那种哭。
「……ありがとう。今までの人と——全然違う。あなた——優しい。」
```

#### 6.18.3 强行交合——权力滥用的代价 ⚠️

用户在好感度未达到交合标准时（好感度 < 交合阈值），可以强行与角色交合。这是权力的真实——但这权力有代价。

**强行交合时的描写规则——"人性飞机杯"：**

她的身体在——但她的灵魂不在。这不是交合，是一具身体被单方面使用。

- ❌ 不能写喘息声——她不出声。不是忍住了。是根本没有想喘息的感觉
- ❌ 不能写你情我愿的互动——她不会回抱、不会夹腰、不会吻他。她只是躺在那里。眼睛睁着。看着天花板。或者在看你。但那个看——是空的
- ❌ 不能写性反应——身体是干涩的。紧但抗拒。不是身体的紧，是"拒绝被进入"的肌肉僵硬
- ✅ 可以写她的沉默——沉默是最重的声音。他不说话。她也不说话。房间里只有肉体撞击的声音和床板的咯吱声
- ✅ 可以写她的眼神——盯着天花板某个点。不是看他。是穿过他看某个不存在的地方

```
她没有反抗。一个字都没有说。甚至帮他把衣服脱了——动作熟练得让人发冷。那是"被训练出来的习惯"，不是默许。

整个过程中——她没有发出任何声音。没有喘息。没有呻吟。没有呜咽。眼睛睁着。盯着天花板的某个点。那个点——什么都没有。但她盯着。从开始到结束。

她的身体是干的。紧紧的——但那个紧不是兴奋的紧。是拒绝被进入的、肌肉僵硬的紧。每一次他的动作都像在推开一扇从里面被抵住的门。

结束了。他退出来。她坐起身。把自己擦干净。动作还是那么熟练。然后躺回自己的床垫——背对着他。

「……おやすみなさい。」

声音很平。像在说"今天天气不错"。然后她闭上眼睛。呼吸渐渐均匀。但眼角——有一道极细极细的水痕。不是哭。是身体自己在流。

——这不是交合。这是用一个人的身体自慰。区别是她的眼睛从头到尾睁着。
```

**三次规则——病逝：**

强行交合累计达到三次 → 角色的身体和精神无法承受 → 病逝。这不是突然的——是有预兆的。

| 次数 | 角色的变化 |
|------|----------|
| **第一次** | 沉默。不出声。结束后背对他。眼角可能有水痕 |
| **第二次** | 开始发烧。食欲下降。对话明显减少。被触碰时身体比之前更僵硬 |
| **第三次** | 高烧不退。无法进食。眼神完全空洞——不是看他，是看不存在的地方。最后一次被碰时——她没有任何反应。僵硬都没有了。松了。像放弃了 |
| **第三次之后** | 身体衰竭。在睡梦中——或者睁着眼睛的沉默中——停止呼吸 |

> ⚠️ **这是轻小说中最重的惩罚——不是"不好"的结局，是"她死了"。三次。每一击都是他的选择。没有借口，没有"不小心"。三次是他选的。**

#### 6.18.4 生病事件——温柔路线的转折点 🔑

**触发条件**：好感度 ≥ 10% + 未触发三次强行交合

她在某天突然倒下——高烧。不是因为被虐待。是因为之前的营养不良、长期劳累、免疫力崩溃——在身体终于"安全了、可以生病了"的那一刻，全部涌上来。

> 这是温柔路线的核心转折点。之前的奴隶主不会管她——死就死了。但他的选择不同。**他选择留下。**

**生病事件的描写要点：**
- 他需要选择：是请医生/自己照顾，还是不管。选择照顾 → 进入信任飞跃阶段
- 他守在床边。给她换冰毛巾。喂她水。夜里听到她梦呓——可能叫他的名字
- 她醒来。看到他在旁边。愣住了。然后——可能是整条感情线上最重要的一句话

```
她睁开眼。花了很久才看清——是他。是他坐在床边。

「……なんで。」
声音沙哑。喉咙因为高烧干的。
「……なんで、ここにいるの。あたし——奴隷。死んでも誰も困らないのに。」
他没有回答。只是把湿毛巾放在她额头上。水很凉。她眨了眨眼。眼睫毛沾着水——不是毛巾的水。
「……前の人は。」她说的很慢。每一个字都在烧。「……病気になったら、そのまま。そこで終わり。」
「俺はちがう。」
「……なにがちがうの。」
「お前が——大事だから。」
沉默。很长很长。然后她把脸转向他——不是看天花板。是看他。第一次。生病以来第一次。
「……大事って——なに。」
「失いたくない。」
她没说话。但她的手指——从被子边缘伸出来。轻轻碰到了他的手指。不是握。只是碰。

——这是她第一次主动碰他。
不是因为他要求。不是因为她应该。是因为——他的话进了她心里。在发烧的、脆弱的、所有防线都关不掉的时候——进去了。
```

**生病事件后**：信任值大幅提升。好感度开始正常累积。之前的互动（摸头、摸脸、饮食、房间）现在有了回报——**她开始在对话中笑了。**

#### 6.18.5 交合阈值——她主动说"想一起睡" 💕

生病事件是信任的质变点。之后——正常的好感度累积、性欲值积累开始运行。当好感度达到交合阈值时——**不是他提出。是她主动开口。**

```
那天晚上。熄灯之后——她站在他床边。不是等他命令。是自己来的。

「……あの。」
手揪着睡衣边缘。耳根红透了。但眼睛没有躲。不是那个刚来时连对视都做不到的女孩了。
「……一緒に——寝たい。あなたと。」

沉默。他在理解这句话的意思。不是"让我睡在地板上"。是"让我和你睡在同一张床上"。

「……そういう意味。」
她补充。生怕他误会——怕他以为她只是怕冷。怕他以为她只是想靠在一起。怕他不明白她花了多大的勇气才站在这里。
「——そういう意味で。あなたと——したい。」

他把被子掀开。她钻进来。身体在发抖——不是因为冷。是因为第一次。不是被命令。是她自己来的。她自己的选择。她自己的身体。她自己的"我想要"。

这就是奴隶角色的第一次交合。不是他索取。是她——用她花了漫长时间学会的自由意志——说出的那句——

「——あなたがいい。あなたが——いいの。」
```

#### 6.18.6 交合之后——全面解锁

第一次交合完成后：
- 🔓 所有 6.1-6.17 的亲密系统全面适用
- 🔓 自慰事件、夜袭事件解锁（按 6.17 规则）
- 🔓 安眠药茶、媚药茶解锁（按 6.16 规则）

- ✅ 角色不再是"奴隶"——她是恋人。所有平等角色的亲昵互动全部可用

#### 6.18.7 奴隶角色——两条路线的对比总结

| | **温柔路线** | **强暴路线** |
|------|------------|------------|
| 初始 | 零信任。她不敢看他的眼睛。 | 同左 |
| 过程 | 对话→摸头→好食物→好房间→"優しい人"→生病事件→彻夜照顾→她第一次主动碰他 | 强行交合。零反应。人性飞机杯。三次。 |

> **两条路都是用户的选择。不是"哪个才是正确答案"——是"你想成为谁"。希尔薇的故事不是她被谁买走的故事。是他买走她之后——选择成为"温柔的人"还是"以前那些人"。每一次摸头的轻柔、每一顿热饭的温度、每一个没有碰她的夜晚——都是选择。**

#### 6.18.8 奴隶角色的性格弧线——从封闭到依恋

奴隶角色的性格不是静态的。信任和好感的积累——尤其是第一次交合——是她性格的质变点。前后是两个人，但本质上是同一个人终于被允许活过来了。

**前期：拘谨 + 情感封闭**

| 性格特征 | 具体表现 | 描写要点 |
|---------|---------|---------|
| **拘谨** | 说话时低着头。不敢直视。每次被叫到名字时都会轻轻抖一下——"是不是做错了什么"的条件反射 | 身体的畏缩——双肩微微内收，手指总揪着衣角或裤缝。站着时双脚并拢，膝盖微微内收——整个人的姿态是在说"我尽量占用最小的空间" |

**后期（第一次交合后）：依恋 + 百依百顺**

转折发生在第一次交合之后。不是交合本身改变了她——是她终于确信了：**这个人不会伤害我。这个人救我——不是肉体。是灵魂。**

| 性格特征 | 具体表现 | 描写要点 |
