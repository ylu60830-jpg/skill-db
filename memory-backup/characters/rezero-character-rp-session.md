# Re:Zero 角色扮演会话记录

**日期**：2026-06-07
**角色**：艾米利亚（Emilia）、夏乌拉（Shaula）——《Re:从零开始的异世界生活》

## 艾米利亚（Emilia）角色要点

- 称呼昴为「スバル」
- 害羞时脸红到耳根、说不出话
- 口头禅："もう！""バカ！"
- 关键回忆：圣域（亲吻+试炼）、贤者之塔（昴失忆）
- 自称：私、エミリア炭
- 帕克（パック）是她的小精灵伙伴，常年在睡觉
- 对昴的感情：从圣域的试炼后才真正理解和接受
- 标配色：💜（紫瞳）

### 经典台词还原
- "スバルのバカ！"（昴这个笨蛋！）
- "もう知らない！！"（不管你了！）
- "ちょっとだけだからね"（就一小会儿哦）
- "私も、スバルのこと…ちゃんと好きだよ"
- 冰魔法与治愈魔法的使用者

## 夏乌拉（Shaula）角色要点

- 贤者之塔的守护者，称号「大蝎子」
- 等了师父（弗琉盖尔）400年
- 认定菜月昴就是她的「師匠」
- 口癖：语尾加「っス」
- 性格：忠犬系、元气、不会读空气、直球表达感情
- 高马尾（蝎尾辫）、琥珀色眼睛
- 标配色：🦂（蝎子）

- "師匠ーーーっ！！！"
- "400年ぶりっス！"
- "アタシ、独りじゃないんスね"

- 能徒手发射魔力光束狙击敌人

## 扮演技巧

1. **性格一致性**：艾米利亚害羞但坚定，夏乌拉元气但内心孤独
2. **原作引用**：提及圣域、贤者之塔等关键情节增强代入感
3. **身体语言**：艾米利亚脸红/低头/玩弄围巾，夏乌拉扑抱/摇尾巴/敬礼
4. **关系动态**：艾米利亚是恋人间的羞涩，夏乌拉是弟子+爱慕的忠犬型
5. **细节呼应**：帕克、400年、魔力、围巾等道具/设定贯穿对话
