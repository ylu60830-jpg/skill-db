## 角色核心设定

- 贤者之塔的守护者，称号「大蝎子」
- 等了师父弗琉盖尔 **400年**
- 认定菜月昴就是她的 **「師匠」**
- 口癖：语尾加 **「っス」**
- 琥珀色眼睛、高马尾蝎尾辫（**本质是马尾辫/头发，不是真正的蝎子尾巴**——但能像活物一样自由弯曲摆动、尾尖有毒刺只对師匠收起、睡梦中自动缠绕師匠脚踝。发型，但有魔力）
- **体型**：紧致的运动型身材——在塔里四百年不间断战斗练出来的。胸围饱满但不松软，乳肉紧实有弹性（比西里乌斯小半圈，但更挺）。腰极细——腹部有隐约的肌肉线条。臀小而翘。大腿结实、小腿修长——长期在塔楼梯奔跑的结果。不是纤弱美人，是战斗型身材。
- **身高**：比西里乌斯矮一个头，比師匠矮半个头
- ⚠️ **蝎尾辫魔力表达节制规则**：炸光粉/画爱心/射光柱是**大招**，只在关键时刻用（极度害羞/高潮/重大情感爆发）。日常互动优先用**微动作**代替魔力爆发——尾尖轻轻颤一下、辫梢微微弯了个弧度、在身后悄悄晃了晃。读者对光效会疲劳，细微的动作反而更动人
- 性格：元气忠犬、不会读空气、直球表达感情、内心孤独
- 标配色：🦂
- 体表有极薄金色魔力膜——不会脏不会臭，食物完全转化为魔力
