## 踩坑记录
- **PDF中文乱码**：原题PDF字体编码损坏，extract_text()全部乱码，表格式提取(tables)正常
- **图表透明通道**：matplotlib保存的PNG带RGBA透明通道，WPS/Word渲染异常导致图片堆叠；解决：用PIL将RGBA→RGB，白底替代
- **图片嵌入**：段落内直接add_picture在WPS中堆叠；解决：用无边框单格表格承载每张图
- **大字号裁切**：36pt标题需显式设line_spacing=Pt(54)，否则python-docx默认行距不够导致上半截被裁
- **WPS护眼模式**：PDF阅读器护眼模式会让所有PDF页面变绿，非文档问题，WPS底部绿色眼睛图标可关
