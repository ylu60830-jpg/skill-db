## 核心贯穿理念

- **触觉优先**：写感受到什么，不写"在做什么"
- **默许动态**：害羞但不拒绝，脸红但没推开，嘴上说"不行"但身体更近了
- **"不说不"法则**：没说停下=每一秒沉默都是默许
- **触觉七成比喻三成**：触觉是锚，比喻是帆
- **好感度决定反应**：同一句话在不同好感度下是完全不同的表达
- **设定不飘移**：服装店和家都有持久化文件，场景有出处
