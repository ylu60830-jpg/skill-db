# 游戏大厅 — 设计方案

## 项目概述

一个经典的像素风（Pastel Pixel）游戏集合网页，包含 4 款可玩游戏。纯 HTML/CSS/JS 实现，个人展示 + 前端练手项目。

## 技术栈

- 纯原生三件套：HTML + CSS + JavaScript
- 零外部依赖，零框架
- 部署：GitHub Pages 或本地静态服务器

## 视觉风格：Pastel Pixel

星露谷式治愈像素风——柔和粉彩 + 像素方块感。

```css
/* 色板 */
--bg:       #f5f0e8   暖米色背景
--yellow:   #ffd5c2   扫雷卡片（柔和蜜桃）
--blue:     #c2d5ff   俄罗斯方块卡片（柔和蓝）
--green:    #c2ffd5   贪吃蛇卡片（柔和薄荷绿）
--pink:     #ffc2f0   2048 卡片（柔和粉）
--brown:    #6d5c4a   主文字（可可棕）
--border:   #d4c5b9   边框色
```

- 字体：标题 "Press Start 2P"（Google Fonts 像素字体），正文 monospace 兜底
- 边框 3px solid，直角无圆角
- box-shadow: 4px 4px 0 像素硬阴影
- image-rendering: pixelated

## 游戏规格

### 💣 扫雷（最爱 ⭐）

基于 Windows 经典扫雷规则。

| 难度 | 网格 | 雷数 | 密度 |
|------|------|------|------|
| 🟢 初级 | 9×9 | 10 | ~12% |
| 🟡 中级 | 16×16 | 40 | ~16% |
| 🔴 高级 | 30×16 | 99 | ~21% |

- 首次点击安全
- 左键翻开 / 右键标旗 / 再右键切换问号 / 双键 Chord
- 数字 0 泛洪自动展开
- 计时器 + 剩余雷数计数器
- DOM 格子渲染（方便 CSS 像素风样式）

### 🧱 俄罗斯方块

- 场地 10×20，7 种方块（I J L O S T Z）
- 7-bag 随机系统
- 计分：单消 100 / 双消 250 / 三消 400 / Tetris 1000
- 每 10 行升一级，速度递增
- Canvas 渲染（60fps）

### 🐍 贪吃蛇

- 20×20 格子场地
- 方向键控制，90° 转向，不可掉头
- 食物随机生成，吃后蛇身 +1
- 撞墙/撞自己 = 游戏结束
- 每吃一个 +10 分，速度随分数递增
- Canvas 渲染

### 🔢 2048

- 4×4 格子
- 90% 出 2，10% 出 4
- 相同数字滑动合并，每格每步只合并一次
- 分数 = 合并后新瓦片的值
- 达到 2048 胜利，可继续
- DOM 格子渲染（CSS 数字颜色渐变）

### 游戏页统一结构


| 游戏 | 操控 |
|------|------|
| 扫雷 | 鼠标（点击/右键） |
| 俄罗斯方块 | 键盘（↓←→ 移动，↑ 旋转，空格 硬降） |

| 2048 | 键盘（方向键）+ 触屏滑动 |
